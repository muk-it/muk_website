MuK Transparent Navigation Menu

Website Navigation Menu Style

MuK IT GmbH - www.mukit.at

Overview

Adds a option to the website customize menu to trigger the transparent navigation menu. The menu is fixed to the top of the website and will switch to a non transparent mode while scrolling. In mobile view the menu is in a non transparent mode by default.

Mobile View

On mobile devices the navigation bar isn't transparent and has the color defined in the variables.less file.

Settings

Change the color scheme by editing the less file through the HTML/CSS editor. The variables.less file is located in the static folder of the module.
